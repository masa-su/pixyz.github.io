Let's implement deep generative models more easily

A library for developing deep generative models

Features

Pixyz is a high-level deep generative modeling library, based on PyTorch. It is developed with a focus on enabling easy implementation of various deep generative models.

Intuitive

Pixyz enables users to implement deep generative models (VAE, GAN, etc.) intuitively, as if writing the formulas on paper.

Concise

Pixyz APIs provide independent definitions for deep neural networks, probablistic models and optimizers. This enables users to implement even for complex models concisely.
